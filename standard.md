# Coding Standards and Style Guides

### Naming Convention

- **Go File Names, Package Names, Public Methods, Public Members, Public Package Functions and Global Variables** : `PascalCase`
- **Global Constant Variables** : `ALL_CAPS_SNAKE_CASE`
- **Private Members and Methods** : `camelCase`
- **Private Package Functions and Local Variables** : `snake_case`
- **Boolean Variables** : should start with `is/Is/IS_` or `has/Has/HAS_`

### Terms and Definitions

1. **Public Package Functions** - are functions inside a package that is also accessible to other package.

2. **Private Package Functions** - are functions inside a package that is only accessible to that package.
